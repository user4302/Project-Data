**Project Title:** Grafana Monitoring and Alerting System

**Type:** Professional

**Short Description:**
This project implements a comprehensive monitoring and alerting solution using Grafana, Prometheus, and Blackbox Exporter. The system provides real-time visibility into system performance metrics, website availability, and service health monitoring. It enables proactive infrastructure monitoring through customizable dashboards and automated alerting for critical system thresholds.

**My Role & Contribution:**
I designed and implemented a complete monitoring stack including Prometheus for metrics collection, Grafana for visualization, and Blackbox Exporter for external service monitoring. I created multiple pre-configured dashboards for system metrics, website monitoring, and transaction tracking, along with alert rules for CPU and memory usage thresholds. I also configured monitoring targets for multiple external websites and set up the complete export pipeline.
