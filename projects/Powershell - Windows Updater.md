**Project Title:** Windows Update Tool

**Type:** Open-Source

**Short Description:**
A PowerShell-based menu-driven tool for managing Windows updates with a clean, user-friendly interface. The tool provides a simple way to check for, review, and install Windows updates with options for automatic reboot management. It leverages the PSWindowsUpdate module to provide comprehensive update management capabilities through an intuitive text-based menu system.

**My Role & Contribution:**
I designed and implemented a complete Windows update management tool with an interactive menu system. I built the core functionality including update checking with formatted table display, selective installation options, reboot management, automatic elevation handling, and dependency management for the PSWindowsUpdate module. I created a user-friendly interface with proper error handling and clear output formatting.

**Key Technologies/Skills:**
- PowerShell 5.1+ scripting
- Windows Update API integration
- PSWindowsUpdate module management
- System administration and privilege elevation
- Menu-driven interface design
- Error handling and logging

**Impact/Results:**
- Simplified Windows update management through an intuitive interface
- Automated dependency installation (PSWindowsUpdate module)
- Reduced complexity for non-technical users managing system updates
- Provided granular control over reboot behavior during updates
- Enhanced user experience with formatted output and clear progress indicators
